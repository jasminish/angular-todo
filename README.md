# Angular-Todo 
A sample todo-application with user authentication system built on [AngularJS](https://angularjs.org) and [UI-Router](https://ui-router.github.io/).  
All data is saved to local storage / session storage. 

To run:  
``` 
$ npm install  
$ npm start 
```
